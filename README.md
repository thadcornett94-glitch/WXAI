## 🚀 Getting Started

### Prerequisites
- Node.js 14+
- npm or yarn
- Gemini API key
- Audio processing tools (FFmpeg)

### Installation

```bash
# Clone the repository
git clone https://github.com/thadcornett94-glitch/WXAI.git
cd WXAI

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Add your GEMINI_API_KEY and settings

# Start development server
npm run dev

# Access at http://localhost:3000
```
